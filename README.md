# Agenda Qt

Desktop application written in C++ and Qt for managing personal activities such as events, deadlines, reminders and checklists.

## Features planned

- Activity creation, editing, deletion and visualization
- Advanced search and filters
- JSON persistence
- CSV import/export
- Recurring activities
- Templates
- Undo/redo
- Qt Widgets GUI with custom QSS styling

## Technologies

- C++
- Qt 6
- CMake
